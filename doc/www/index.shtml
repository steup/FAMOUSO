Overview

FAMOUSO is a publish/subscribe middleware that readopts, substantiates and refines the ideas and concepts of its predecessor COSMIC (COoperating SMart devICes). Additionally, FAMOUSO provides much more features like adaptation to a specific platform or the generic gateway architecture.

As communication model, the middleware supports event-based communication according to the publisher/subscriber concept. FAMOUSO is especially designed to allow cooperation between smart sensors and actuators on different hardware platforms ranging from 8-bit micro-controllers up to 32-bit PC/Workstations. It supports interaction over a broad variety of communication media like Controller Area Network, 802.15.4, AWDS, UDP-MultiCast. On application level, it offers a unique API that is easy to use from different programming languages (C/C++, Python, Java, .NET) as well as from engineering tools (Matlab/Simulink, Labview).

Key features of FAMOUSO are:
  • Publish/Subscribe middleware
  • Uniform addressing of topics
  • No control flow dependancies between Publisher and Subscriber
  • Abstraction of the concrete underlying communiction media
  • Hides heterogeneity
  • Portability
  • Adaptability
  • Configurability
  • Deployment of modern software technologies
