Download Page

You have two options:

  1. you can download an official release from the FAMOUSO download page at Sourceforge

  2. or you can checkout the last state of the development from the FAMOUSO subversion repository.

    Use the following command
    svn co https://svn-eos.cs.uni-magdeburg.de/repos/staff/mschulze/Research/MIKRO/famouso
    This grants only read access to the repository and it is not always warranted that all sources will compile or work. This is due to the fact, that the system is under active development.
