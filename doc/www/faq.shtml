What is FAMOUSO?

FAMOUSO (short for Family of Adaptive Middleware for autonomOUs Sentient Objects) is an open-source effort to develop a quality event-based publish/subscribe communication middleware written in C++ that is able to run on a broad variaty of platforms ranging from 8Bit micro-controllers up to 32Bit PC/Workstations.

On which platforms will FAMOUSO run?

FAMOUSO currently runs on all flavors of *nix and Windows and on at90can128 AVR from ATMEL.

When will FAMOUSO be done?

First you need to define what FAMOUSO should be capable of when it is to be considered done. Since this definition may vary greatly between people, this question cannot be completely answered. One thing is for sure: FAMOUSO is under active development and it will continue to be developed, as there will always be a need for improvements. For more information, visit the /development.shtml">development page and further pages for open issues and feature requests.

How is FAMOUSO licensed?

FAMOUSO is licensed under a single unified /LICENSE.txt">license. This so-called modified Berkeley license is intented to be compatible with most Free Software licenses like the GPL, yet impose as little restrictions for the use of the library in closed-source commercial applications as possible.
